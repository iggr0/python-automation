# Python Automation & Programming Fundamentals

**Source:** *Automate the Boring Stuff with Python*

## Roadmap and Phases

### Phase 1: Syntax and Data Structures

* **Chapters 1, 2 & 3 (Basics, Flow Control, Functions):** Základná syntax Pythonu, vetvenia kódu (podmienky, cykly) a definície funkcií bez nutnosti manuálnej správy pamäte.
* **Chapter 4 (Lists) & Chapter 5 (Dictionaries):** Práca s dynamickými poľami a kľúč-hodnota štruktúrami pre spracovanie štruktúrovaných dát a JSON formátov.
* **Chapter 6 (Strings):** Využitie vstavaných metód na úpravu, čistenie, delenie a formátovanie textových reťazcov.

### Phase 2: Process Automation

* **Chapter 7 (RegEx):** Vyhľadávanie textových vzorov a hromadná extrakcia špecifických dát (napr. sieťových adries, e-maily alebo hashe) z neštruktúrovaného textu.
* **Chapter 8 (File I/O):** Správa lokálneho súborového systému, otváranie, zápis a efektívne čítanie veľkých dátových súborov po riadkoch s ohľadom na optimalizáciu pamäte.
* **Chapter 11 (Web & API):** Komunikácia cez sieť pomocou HTTP požiadaviek (GET/POST), posielanie autentifikačných tokenov a natívne spracovanie sieťových odpovedí vo formáte JSON.

### Phase 3: AI Integration and Advanced Workflows

* **Programmatic LLM Control:** Využitie vývojárskych balíkov (SDK) na posielanie dát, logov alebo textov priamo do jazykových modelov (Claude/OpenAI) na hĺbkovú analýzu.
* **Structured AI Logic:** Nastavenie promptov tak, aby model vracal odpovede v čistej dátovej forme (JSON). To umožňuje skriptu automaticky vyhodnocovať závery z AI a vykonávať na ich základe ďalšie programové akcie.
* **System Orchestration:** Prepojenie vytvorených Python skriptov s externými službami do komplexných, automaticky spúšťaných tokov (Playbookov).
